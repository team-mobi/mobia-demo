## mobia-demo

`mobia`는 mobi + utopia의 합성어로 mobi에서 프로젝트를 처음 진행하는 인원들에게 목표에 맞는 이상적인 개발 경험할 수 있도록 돕기 위해 제작 된 프로젝트입니다

### Getting Started

```
npm install
npm start
```
